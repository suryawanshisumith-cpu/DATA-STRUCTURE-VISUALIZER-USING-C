# Data Structure Visualizer

A simple mini project made using C programming to understand and visualize basic Data Structures operations through menu-driven programs.

This project demonstrates how different data structures work internally using simple implementations and console-based output.

## Features

* Stack Operations

  * Push
  * Pop
  * Display

* Queue Operations

  * Enqueue
  * Dequeue
  * Display

* Linked List Operations

  * Insert
  * Delete
  * Display

* User-friendly menu system
* Simple and beginner-friendly code
* Console-based visualization

## Technologies Used
* C Programming
* Data Structures
* File Handling (optional)
* GCC Compiler

## Project Objective
The objective of this project is to improve understanding of fundamental data structures and their operations through practical implementation in C.

## How to Run
1. Download or clone the repository
2. Compile the program
3. Run the program


## Learning Outcomes
* Understanding of Stack, Queue, and Linked List
* Menu-driven programming in C
* Dynamic memory concepts
* Problem-solving using Data Structures
